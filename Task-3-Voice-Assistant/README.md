# Task 3: Voice Assistant

## Description
This project is a basic Voice Assistant built using Python.
It listens to voice commands and performs simple tasks like greeting,
telling the current time and date, and searching the web.

## Features
- Responds to "Hello"
- Tells current time
- Tells current date
- Performs web search
- Exit command

## How to Run
1. Install required libraries:
   pip install speechrecognition pyttsx3 pyaudio
2. Run the program:
   python voice_assistant.py

## Concepts Used
- Speech recognition
- Text to speech
- Date and time
- Web browser automation
