# Task 2: Random Password Generator

## Description
This project is a command-line Random Password Generator written in Python.
It generates a secure password based on user-defined length and character preferences.

## How to Run
1. Open terminal or command prompt
2. Navigate to the project folder
3. Run:
   python password_generator.py

## Sample Output
Enter password length: 8  
Include letters? yes  
Include numbers? yes  
Include symbols? no  
Generated Password: aB3kP9Qm

## Concepts Used
- Python modules (random, string)
- User input
- Conditional statements
- Loops
