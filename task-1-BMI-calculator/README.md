# Task 1: BMI Calculator

## Description
This project is a command-line BMI Calculator written in Python.
It takes weight (in kilograms) and height (in meters) as input,
calculates the BMI, and displays the BMI category.

## How to Run
1. Open terminal or command prompt
2. Navigate to the project folder
3. Run:
   python bmi_calculator.py

## Sample Output
Enter your weight in kilograms: 70  
Enter your height in meters: 1.75  
Your BMI is: 22.86  
Category: Normal weight

## Concepts Used
- Python variables
- User input
- Arithmetic operations
- Conditional statements (if-elif-else)
